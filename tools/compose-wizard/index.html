Welcome to the Yuzu Compose Wizard

This wizard will walk you through creating a customised docker-compose.yml and .env file for your Yuzu deployment.

Why use this? The default compose file works, but it has hardcoded values and several known gotchas. This wizard helps you avoid common mistakes and generates a production-ready configuration.

What you'll configure:

  • Yuzu version, admin credentials, and ports
  • Data storage and ClickHouse settings
  • Prometheus & Grafana observability stack
  • Yuzu Gateway (Erlang/OTP) settings

What you'll get:

  • docker-compose.yml — fully commented, ready to run
  • .env — environment variables for easy reconfiguration
  • Quick-start commands to get going immediately

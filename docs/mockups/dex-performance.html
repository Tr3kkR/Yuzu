Cohort benchmarking
The Aternity model: fleet-relative percentiles per cohort, so a model (or image) that runs hot stands out. Δ pills compare the cohort p50 to the fleet p50. Read within a column. Cohorts are the distinct values of an operator-chosen tag key — hardware-model cohorts come from the asset-tagging recipe (tag the fleet once via REST/MCP), not a central inventory column.

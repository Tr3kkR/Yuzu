Yuzu places agentic colleagues inside your endpoint estate — reasoning over Windows, Linux, and macOS in real time. Not a console. Not a script runner. Something that was not here before.

the shape of it

One control plane. Three layers.

Operators speak to the Server. The Server reasons, targets, and dispatches. The Agent executes on the endpoint over mTLS. Every edge below is exactly one cell wide — and the build refuses to ship if it isn't.

Scope walking

Yuzu's differentiator: a query's result set becomes a first-class targeting primitive. You walk from one answer into the next — from_result_set:<id> composes with tags and OS filters, and every hop is recorded in the audit chain.

instruction · yaml
# target every host that failed last night's patch sweep
scope:
  from_result_set: "patch-sweep-2026-06-04"
  where: "os:windows AND NOT tag:quarantined"
instruction: software.patch.apply
  timeout: 3600

Scope Engine

The Scope Engine resolves an expression tree — AND / OR / NOT over tags, OS, hostname, arch, agent version, and management groups — into the concrete set of devices an instruction targets.

Active link gets a cyan spine and a lifted ink; hover is a 6%-cyan wash. Quiet, legible, unmistakably the same world as the hero.
